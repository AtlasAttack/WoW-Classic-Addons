# WoW Classic Addons
Addons for World of Warcraft - Classic _(patch 1.12.1)_

## Addon list
* Atlas World Map
* Deadly Boss Mode
* Gatherer
* One Bag
* Questie
* Recount
* Sexy Map
* X and Y

## Instalation
* Just copy folder _Addons_ to your "_{path}/World of Warcraft/\_classic\_/Interface_"
* In game just turn on/off the addons you need
